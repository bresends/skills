# Tag Processes Workflow

Steps for tagging/classifying processes in SEI.

## Prerequisites
- Active SEI session (run `login` first)
- Process ID and desired tag/classification

## Steps
1. Navigate to process page
2. Open classification panel
3. Apply tag based on rules (type, unit, subject)
4. Confirm and save
