# Extract PDF Workflow

Steps for downloading full process PDFs from SEI.

## Prerequisites
- Active SEI session (run `login` first)
- Process ID

## Steps
1. Navigate to process page
2. Locate "Generate PDF" or document download option
3. Handle multi-document processes (merge or download individually)
4. Save to local directory
